# Project Roadmap

## Phase 0 – Initialization
- Define scope
- Design architecture
- Establish repository structure

## Phase 1 – Extract
- Download dataset for given year/month
- Validate file existence
- Record row count and metadata

## Phase 2 – Transform
- Apply data cleaning rules
- Standardize schema
- Output cleaned parquet

## Phase 3 – Load
- Load data into DuckDB
- Create tables
- Provide example analytical queries

## Phase 4 – Refinement
- Add CLI interface
- Add logging
- Add minimal tests
- Improve documentation
